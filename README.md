## 📂 Project Index

### 1. Log Analysis & Security Event Investigation

Folder: `01-log-analysis-lab/`  

A lab focused on analyzing authentication and system logs to identify suspicious activity and document an incident from detection to recommendation.

**Highlights:**

- Investigate failed and successful login patterns  
- Correlate events across multiple log sources  
- Produce an incident-style write-up with timeline and remediation steps  

**Skills:** Log analysis, incident triage, written reporting

---

### 2. Vulnerability Assessment & System Hardening

Folder: `02-vuln-assessment-lab/`  

A small lab environment where I identify vulnerabilities and misconfigurations on Windows/Linux hosts, then propose hardening steps.

**Highlights:**

- Enumerate system configurations and services  
- Prioritize findings based on risk  
- Recommend practical remediation and security baselines  

**Skills:** Vulnerability assessment, system hardening, risk-based thinking

---

### 3. Python Log Parser for Suspicious Activity

Folder: `03-python-log-parser/`  

A simple Python script project that parses log files to flag potentially suspicious events and output a human-readable summary.

**Highlights:**

- Search for failed logins, privilege changes, and other patterns  
- Summarize key details (user, IP, timestamp, event type)  
- Serve as a foundation for future automation and enrichment  

**Skills:** Python, automation, log parsing
